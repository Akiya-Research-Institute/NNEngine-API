Parameters
inputTextureOriginal image
outputHxWxBGRThe image after affine transform as an array of bytes whose length is (Height x Width x 3)
inverseNormalizedAffineMatThe matrix of inverse of the affine transform. The scales of original and destination images must be normalized to (1, 1). This is equivalent to the matrix which represents the affine transform in the UV space.
doFlipImageWhether to flip the image during resizing.
doRotateImageWhether to rotate the image during resizing.
